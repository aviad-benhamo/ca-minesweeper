# Security Policy

## Supported Versions

This repository contains a small static browser game. Security updates are
provided for the latest version available on the default branch.

| Version | Supported |
| --- | --- |
| Latest default branch | Yes |
| Older snapshots or forks | No |

## Reporting a Vulnerability

Please report security vulnerabilities privately by opening a GitHub security
advisory or by contacting the repository owner through the GitHub profile.

Do not open public issues for suspected vulnerabilities until the issue has
been reviewed and a fix or disclosure plan is available.

## Response Expectations

- Initial acknowledgement: within 7 days.
- Initial assessment: within 14 days.
- Fix or mitigation plan: based on severity and project scope.

## Disclosure

Responsible disclosure is appreciated. Public disclosure should happen only
after the vulnerability is confirmed, a fix is available, or a coordinated
timeline has been agreed.
